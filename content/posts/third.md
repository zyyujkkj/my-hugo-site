要创建一篇新的博客，首先需要回到根目录：
```bash
cd ~/my-site
```
然后执行：
```bash
hugo new posts/third.md
```
然后便可以用vim修改：
```bash
vim content/posts/third.md
```
或者用VS Code打开修改：
```bash
code content/posts/third.md
```
